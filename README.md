## Project Description
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset
to understand sales trends, patterns, and customer purchasing behavior.
Python libraries are used to clean, analyze, and visualize the data.

## Dataset
The dataset contains retail sales information such as:
- Order date
- Product details
- Sales amount
- Quantity sold

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Data loading and cleaning
- Handling missing values
- Descriptive statistics
- Time series analysis of sales
- Product and customer analysis
- Data visualization (bar charts, line plots)
- Business insights and recommendations

## Results
The analysis provides useful insights into sales performance and trends,
which can help businesses make data-driven decisions.
